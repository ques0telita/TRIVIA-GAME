# Trivia Difícil 🧠
¡Bienvenido! Este es un proyecto de una aplicación de trivia interactiva desarrollada desde cero. El objetivo principal fue crear una experiencia visualmente atractiva utilizando un fondo de video dinámico y una lógica de juego fluida mediante manipulación del DOM.

## Características principales 
-Interfaz Dinámica: Fondo de video con efecto overlay para mejorar la legibilidad.

-Sistema de Preguntas: Las preguntas incluyen imágenes de referencia que se cargan dinámicamente según el tema.

-Feedback Visual: Los botones cambian de color (verde/rojo) instantáneamente al responder para indicar si acertaste.

-Diseño Adaptable: Los botones de respuesta se ajustan automáticamente en columnas o filas dependiendo del tamaño de la pantalla.

## Tecnologías utilizadas
-HTML: Estructura semántica y contenedores de video.

-CSS: Flexbox para el diseño de botones, posicionamiento fijo para el video de fondo y efectos de transición.

-JavaScript: Lógica del juego, manejo de arreglos de objetos y manipulación de clases en tiempo real.

## Estructura del Proyecto
-index.html: Estructura base y contenedores principales.

-styles.css: Estilos globales, animaciones y diseño responsive.

-index.js: Cerebro del juego (carga de preguntas, validación y control de puntaje).
